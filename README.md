# AudioPlayer  
An Android app decoding audio file to render waveform.  

## Dependency
- Architecture: Android Jetpack(LiveData, ViewModel)
- Dependency injection: Hilt
- Asynchronous threading: RxJava3, RxKotlin
- UI: Material Design
- Test: Mockito
